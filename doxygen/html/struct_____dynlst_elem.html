Detailed Description

This object represents the element of a list. It contains a pointer to the stored data, as well as pointers to the next and previous list elements. The Prev pointer of the first element and the Next pointer of the last element will always be NULL. It also keeps track of basic information about the stored data.
Please, avoid manipulating its members directly or indirectly.
