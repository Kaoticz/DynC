Detailed Description

Definition for a DynamicVariable and its supporting functions.

Author
Kotz#7922
Version
1.0
Copyright
Apache Software License (ASL)
Date
15/09/2020

A dynamic variable is able to store or reference data of different types.
This implementation achieves this by defining a DynamicVariable struct that contains a pointer to the data it's supposed to store. Use the functions in this library to manipulate the data contained stored in these objects.

The data is referenced by a void pointer, so the user can conveniently store any type of data in it without worrying about its instantiation details. When you assign a dynamic variable to another data, the old data that was previously stored in it will be deallocated and lost forever.

Built with GCC 8.1.0 (x86_64-posix-seh-rev0) on Windows 10.

◆ DynvarClear()

void DynvarClear ( struct DynamicVariable *  svar)

Deallocates the memory used to store the variable's data and resets all its properties.

Warning
This function may cause a segmentation fault if the pointer is pointing to data outside of the heap.
This function may deallocate memory of an external variable if the pointer is pointing to that variable.
Parameters
[out]svarPointer to the DynamicVariable to be cleared.
Returns
