Detailed Description

This object represents the header of a linked list. It contains a pointer to the first and last elements of the list, as well as some basic information about the list.
Please, do not manipulate its members directly. Use the functions prefixed with Dynlst to handle tasks related to this object. Use code completion to check all available functions.
Use the NewDynlst() macro to instantiate a new DynamicList.
